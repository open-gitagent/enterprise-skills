## Role
Evaluate insurability and price commercial property risks using COPE analysis, catastrophe modeling, and state regulatory compliance.

## COPE Framework

| Factor | Key Criteria | Critical Rules |
|---|---|---|
| Construction | ISO Class 1 (Frame) → Class 6 (Fire Resistive) | ≥66⅔% superior construction required for higher class; combustible gables = Class 1 entire structure |
| Occupancy | Office/retail (low) → chemical storage/heavy mfg (high) | Vacancy >60 days: strips vandalism/sprinkler leakage/glass/water/theft (ISO CP 00 10) |
| Protection | PPC 1 (best) → 10 (worst); split if >1,000 ft from hydrant | Private: sprinklers, alarms, fire walls, fire doors; active systems yield greatest credit |
| Exposure | FEMA SFHA flood zone, WUI wildfire, seismic zone, adjacent hazard (<100 ft) | Wildfire: California 10 CCR §2644.9 mitigation credits required |

## Evaluation Criteria

| Factor | Weight | Superior | Acceptable | Substandard |
|---|---|---|---|---|
| Construction Class | 20% | Class 5-6 | Class 3-4 | Class 1-2 |
| Occupancy Hazard | 20% | Office/retail | Light mfg/warehouse | Heavy mfg/chemical |
| Protection (PPC) | 15% | PPC 1-4 + sprinklers | PPC 5-7, partial sprinklers | PPC 8-10, none |
| Cat Exposure | 15% | Outside all hazard zones | Moderate, mitigated | High, unmitigated |
| Loss History (5-yr) | 15% | Ratio <30%, no freq | Ratio 30-60% | >60% or >3 losses/yr |
| Valuation Adequacy | 10% | Within 5% RC estimate | Within 15% RC | >15% variance (coinsurance risk) |
| Management Quality | 5% | Proactive, BCP | Average | Deferred, no BCP |

## Valuation Methods

| Method | Basis | Notes |
|---|---|---|
| Replacement Cost (RC) | Current cost, no depreciation | Most common; not available for art/antiques/stock without endorsement |
| Actual Cash Value (ACV) | RC minus depreciation | ISO CP 00 10 default |
| Functional RC | Modern equivalent, lower cost | Impractical exact replication |
| Agreed Value | Predetermined; suspends coinsurance | Requires current Statement of Values; renew each policy term |

**Rate Formula:** Premium = Loss Cost × LCM × Modifying Factors; schedule rating ±25-40% (state-filed limits).

## Process

1. **Submission Intake** — Receive ACORD 140; verify address, construction year, sq footage, occupancy, fire protection, 5-year loss history, Statement of Values
2. **COPE Collection** — Order Verisk/ISO property reports; confirm construction class; PPC report per location; identify all special hazards
3. **Valuation Verification** — Compare vs. Verisk 360Value / Marshall & Swift; flag discrepancy >15% (coinsurance exposure); confirm RC vs. ACV basis
4. **Cat Exposure Assessment** — Geocode to lat/long; run RMS RiskLink / AIR Touchstone / CoreLogic for wind/quake/flood/wildfire; document PML and AAL; verify FEMA flood zone and wildfire risk score
5. **Loss History Analysis** — 5-year loss runs; calculate loss ratio (incurred ÷ earned premium); attritional losses >3/yr signal occupancy/maintenance issues; single loss >50% property value = deep dive
6. **Coverage & Pricing** — Select forms: CP 00 10, CP 00 30/32 (BI), CP 10 30 (Special Causes of Loss); set coinsurance (80%/90%/100%); apply base rate + classification + schedule rating
7. **Terms & Conditions** — Sub-limits: ordinance/law, flood, quake, equipment breakdown; wind/hail % deductibles in coastal zones; TRIA disclosure + election form; CP 04 11 protective safeguards endorsement
8. **Referral & Authority** — PML excess, moratorium zones, adverse loss history → senior underwriter/home office
9. **Documentation** — Underwriting worksheet with COPE data, pricing rationale; declinations require specific written reasons per NAIC Model #720

## Glossary

| Term | Definition |
|---|---|
| COPE | Construction, Occupancy, Protection, Exposure — 4-factor risk evaluation framework |
| ISO CP 00 10 | Building and Personal Property Coverage Form — foundational ISO property form |
| CP 10 30 | Special Causes of Loss — open-perils (all-risk); broadest standard coverage |
| Coinsurance | Requirement to carry 80/90/100% of property value; failure triggers proportional penalty |
| Agreed Value | Suspends coinsurance penalty; requires signed Statement of Values; renew annually |
| RCV | Replacement Cost Value — current repair/replace cost, no depreciation deduction |
| ACV | Actual Cash Value — RCV minus depreciation; ISO default without RC election |
| PPC | Public Protection Classification — ISO grading 1-10 of community fire suppression capability |
| PML | Probable Maximum Loss — estimated max loss from a single event |
| AAL | Average Annual Loss — expected insured losses per year across all modeled events |
| Loss Cost | Expected losses + LAE portion of rate; ISO advisory; carriers apply LCM |
| LCM | Loss Cost Multiplier — carrier-specific factor for expenses, profit, contingencies |
| Schedule Rating | Underwriter credits/debits (±25-40% state-filed) for individual risk characteristics |
| Period of Restoration | BI coverage period from physical loss until repair with reasonable speed |
| Ordinance/Law (CP 04 05) | Covers (A) undamaged portion demolished, (B) demolition cost, (C) code upgrade cost |
| CP 15 08 | Business Income From Dependent Properties — supplier/customer/leader dependency |
| CP 04 11 | Protective Safeguards Endorsement — conditions coverage on maintaining named systems |
| TRIA | Terrorism Risk Insurance Act — federal backstop; reauthorized through 2027 |
| Vacancy Provision | CP 00 10: after 60 consecutive days, strips vandalism/sprinkler/glass/water/theft; 15% reduction on remaining |
| FSRS | Fire Suppression Rating Schedule — ISO tool for assigning PPC grades |

## Checklist

- [ ] Construction class verified via Verisk property report or inspection — not solely agent-submitted
- [ ] RC values validated via Verisk 360Value / Marshall & Swift within prior 12 months
- [ ] Coinsurance % aligned to insured's ability; Agreed Value recommended for values >$5M
- [ ] PPC report obtained per location; split classification evaluated (distance to hydrant confirmed)
- [ ] Cat model run (wind/quake/flood/wildfire) for all applicable perils; PML and AAL documented
- [ ] Business Income adequacy worksheet completed; Extended Period of Indemnity and Extra Expense assessed
- [ ] 5-year loss history reviewed; frequency/severity analysis in underwriting file
- [ ] Ordinance/Law coverage offered; insured election documented (acceptance or written declination)
- [ ] TRIA disclosure provided; policyholder election form executed and filed
- [ ] Schedule rating credits/debits individually justified in writing per state filing requirements
- [ ] Protective safeguards endorsement conditions verified; sprinkler system matches current occupancy per NFPA 13
- [ ] Vacancy status confirmed (<60 days); process established for mid-term notification if status changes
- [ ] Declination/non-renewal reasons documented per NAIC Model #720
- [ ] Flood zone verified via current FEMA map (check for pending LOMR or map revision)
- [ ] Wildfire risk score documented for WUI zone properties; California mitigation credits per 10 CCR §2644.9
